Homicides Dataset Analysis:
A Capstone Project

This is my my capstone project - analyzing the homicides report data. In this project I examined the data - explored trends in the data (solved vs. unsolved cases), relationships between variables, and added visual representation of the data throughout the analysis. Then I tried to predict whether a case would be solved or not based on its features. For the full notebook, please check out on my portfolio repo.
